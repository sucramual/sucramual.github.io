# Marcus' personal website

Personal portfolio and blog. Live at [https://sucramual.github.io](https://sucramual.github.io).

## Tech Stack
- **Static Site Generator:** [Eleventy (11ty)](https://www.11ty.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) with PostCSS
- **Deployment:** GitHub Pages

## Running Locally

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```
   *This starts the local Eleventy server along with the CSS watch script. The site will be available at `http://localhost:8080`.*

3. **Build for production:**
   ```bash
   npm run build
   ```
